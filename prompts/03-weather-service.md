```
다음 기술 스택으로 우리 동네 날씨 서비스를 구축해줘:
- React + TypeScript (Vite)
- shadcn/ui + Tailwind CSS
- Framer Motion

2. 시간별/주간 날씨 예보

4. 도시 검색 및 저장 기능

디자인 요구사항:
- 미니멀하고 모던한 UI
- Glassmorphism 효과
- 부드러운 애니메이션과 트랜지션
- 반응형 디자인

프로젝트 세팅부터 차근차근 진행해줘.
```
