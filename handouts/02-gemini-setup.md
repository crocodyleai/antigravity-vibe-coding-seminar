## 📄 GEMINI.md란?

프로젝트 루트에 위치하는 **AI 설정 파일**입니다.

### 역할

1. **프로젝트 컨텍스트 제공** - AI가 프로젝트를 이해하는 기본 정보
2. **코딩 스타일 가이드** - 원하는 코딩 규칙 명시
3. **기술 스택 정의** - 사용할 라이브러리/프레임워크 지정

### 예시

```markdown
# GEMINI.md

## 프로젝트 개요
쇼핑몰 판매 데이터 분석 대시보드

## 기술 스택
- Python 3.11
- pandas, matplotlib
- Streamlit (웹 UI)

## 코딩 규칙
- 한국어 주석 사용
- type hints 필수
- 함수마다 docstring 작성
```

---

## 🧠 컨텍스트(Context)란?

AI가 대화를 이해하기 위해 참조하는 **정보의 범위**입니다.

### 컨텍스트에 포함되는 것

| 종류 | 예시 |
|------|------|
| 시스템 프롬프트 | GEMINI.md 내용 |
| 대화 기록 | 이전 채팅 내용 |
| 열린 파일 | 현재 에디터에 열린 코드 |
| 선택된 코드 | 드래그로 선택한 부분 |

### 컨텍스트 윈도우

```
┌─────────────────────────────────────┐
│  시스템 프롬프트 (GEMINI.md 등)     │
├─────────────────────────────────────┤
│  이전 대화 기록                      │
│  ...                                │
│  최근 대화                          │
├─────────────────────────────────────┤
│  현재 입력                          │
└─────────────────────────────────────┘
         ↓
    AI 응답 생성
```

## 💾 메모리와 세션

### 세션(Session)이란?
- 하나의 대화 흐름
- 세션 내에서는 이전 대화를 기억
- 새 세션 시작 시 이전 대화 초기화

### 왜 새 세션을 시작할까?

1. **컨텍스트 오염 방지**
   - 긴 대화 → 오래된 정보가 방해될 수 있음
   
2. **새로운 작업 시작**
   - 다른 주제로 전환할 때
   
3. **토큰 절약**
   - 불필요한 기록 제거

### 언제 새 세션?

| 상황 | 권장 |
|------|------|
| 같은 기능 계속 개발 | 세션 유지 |
| 완전히 새로운 작업 | 새 세션 |
| AI가 이상한 답변 | 새 세션 |
| 컨텍스트가 너무 길어짐 | 새 세션 |

```markdown
- (원하는 스타일)
```
