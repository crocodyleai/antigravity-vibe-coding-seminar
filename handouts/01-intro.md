## 🚀 Antigravity란?

**Google Antigravity** (Project IDX)는 Google이 제공하는 클라우드 기반 AI 코딩 환경입니다.

### 주요 특징

| 특징 | 설명 |
|------|------|
| 클라우드 기반 | 브라우저만 있으면 어디서든 개발 가능 |
| AI 에이전트 내장 | Gemini 기반 코딩 어시스턴트 |
| 즉시 시작 | 환경 설정 없이 바로 코딩 |
| 다양한 템플릿 | React, Flutter, Python 등 지원 |

---

## 🖥️ UI 구성 요소

### 1. 파일 탐색기 (좌측)
- 프로젝트 파일/폴더 구조
- 파일 생성, 삭제, 이름 변경

### 2. 에디터 (중앙)
- 코드 작성 공간
- 문법 하이라이팅
- 자동 완성

### 3. AI 채팅 패널 (우측)
- Gemini와 대화
- 코드 생성 요청
- 디버깅 도움 요청

### 4. 터미널 (하단)
- 명령어 실행
- 서버 실행/로그 확인

---

## 🛠️ 첫 프로젝트 생성하기

### Step 1: Antigravity 접속
1. [idx.google.com](https://idx.google.com) 접속
2. Google 계정으로 로그인

### Step 2: 새 프로젝트 생성
1. "Create a new workspace" 클릭
2. "Blank" 템플릿 선택 (또는 원하는 템플릿)
3. 프로젝트 이름 입력
4. "Create" 클릭

### Step 3: 환경 확인
```bash
# 터미널에서 확인
python --version
node --version
```
