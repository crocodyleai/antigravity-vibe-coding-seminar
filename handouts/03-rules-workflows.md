- 룰(Rules)로 AI 행동 제어

---

## 📏 룰(Rules)

AI의 **기본 행동 방식**을 정의합니다.

### 룰이란?

- AI가 항상 따라야 하는 규칙
- 마크다운 파일로 작성
- 파일당 12,000자 제한

### 저장 위치

| 종류 | 위치 | 범위 |
|------|------|------|
| Global Rules | `~/.gemini/GEMINI.md` | 모든 워크스페이스 |
| Workspace Rules | `.agent/rules/*.md` | 현재 워크스페이스만 |

### 룰 파일 예시

`.agent/rules/korean-style.md`
```markdown
# 한국어 스타일

## 활성화
Always On

## 규칙
- 답변은 한국어로 작성
- 코드 주석도 한국어로
- 에러 설명 시 원인과 해결책 함께 제시
```

### 활성화 방식

| 방식 | 설명 |
|------|------|
| **Manual** | @ 멘션으로 수동 활성화 |
| **Always On** | 항상 적용 |
| **Model Decision** | 모델이 상황에 따라 판단 |
| **Glob** | 파일 패턴 매칭 (예: `*.js`, `src/**/*.ts`) |

### 좋은 룰 작성법

| ✅ 좋은 예 | ❌ 나쁜 예 |
|-----------|-----------|
| "변수명은 snake_case 사용" | "좋은 코드 작성" |
| "함수는 20줄 이내로" | "짧게 작성" |
| "에러는 try-except로 처리" | "에러 처리 잘 하기" |

## 🔄 워크플로우(Workflows)

**반복 작업을 자동화**하는 명령 시퀀스입니다.

### 워크플로우란?

- 여러 단계의 작업을 하나로 묶음
- `/워크플로우명`으로 실행
- 마크다운 파일로 작성 (12,000자 제한)

### 저장 위치

| 종류 | 위치 |
|------|------|
| Global Workflows | Customizations 패널에서 + Global |
| Workspace Workflows | `.agent/workflows/*.md` |

### 워크플로우 파일 예시

`.agent/workflows/주간리포트.md`
```markdown
# 주간리포트

주간 판매 리포트를 생성하는 워크플로우입니다.

## Steps

1. 이번 주 총 매출 계산
2. TOP 5 판매 상품 추출
3. 카테고리별 매출 차트 생성
4. 전주 대비 증감률 분석
5. 마케팅 인사이트 3가지 제안
```

### 사용법

채팅창에 `/주간리포트` 입력하면 해당 워크플로우 실행!

> 💡 워크플로우 안에서 다른 워크플로우 호출 가능!
> 예: "Call /시각화" 라고 적으면 해당 워크플로우도 실행

## 🛠️ 스킬(Skills)

AI의 **능력을 확장**하는 기능 모듈입니다.

### 스킬이란?

- 특정 도메인/작업에 특화된 기능
- 외부 도구/API 연동 가능
- 재사용 가능한 기능 단위

### Antigravity 기본 스킬

| 스킬 | 기능 |
|------|------|
| 코드 생성 | 자연어 → 코드 |
| 코드 설명 | 코드 → 자연어 |
| 디버깅 | 에러 분석 및 수정 |
| 파일 읽기/쓰기 | 프로젝트 파일 조작 |
| 터미널 실행 | 명령어 실행 |
| 웹 검색 | 최신 정보 검색 |
