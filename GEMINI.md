# GEMINI.md - AI 에이전트 설정 예시

> 이 파일은 Antigravity가 프로젝트 컨텍스트를 이해하는 데 사용됩니다.

## 프로젝트 개요

이 프로젝트는 바이브코딩 세미나를 위한 실습 환경입니다.

## 기술 스택

- Python 3.11+
- 데이터 분석: pandas, matplotlib
- 웹 서비스: FastAPI 또는 Flask
- API: Open-Meteo (날씨 데이터)

## 코딩 스타일

- 한국어 주석 사용
- 변수명은 영어 snake_case
- 함수/클래스 docstring 필수
- 에러 핸들링 포함

## 주의사항

- API 키가 필요 없는 공개 API만 사용
- 실습용이므로 프로덕션 수준의 보안은 고려하지 않음
- 결과물은 로컬에서만 실행
