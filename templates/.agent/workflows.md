# Workflows

## /분석시작

데이터 파일을 처음 분석할 때 사용합니다:
1. 파일 읽기 및 구조 파악
2. 컬럼 설명 및 데이터 타입 확인
3. 기본 통계 요약 (건수, 평균, 합계 등)
4. 결측치/이상치 체크

## /주간리포트

주간 판매 리포트를 생성합니다:
1. 이번 주 총 매출 계산
2. TOP 5 판매 상품 추출
3. 카테고리별 매출 차트 생성
4. 전주 대비 증감률 분석
5. 마케팅 인사이트 3가지 제안

## /시각화

데이터를 차트로 시각화합니다:
1. 데이터 특성에 맞는 차트 타입 선택
2. 차트 생성 (제목, 레이블 포함)
3. 차트 해석 및 인사이트 설명

## /정리

분석 결과를 정리합니다:
1. 지금까지 분석 내용 요약
2. 핵심 발견사항 3가지
3. 권장 액션 아이템
