# 🚀 구글 Antigravity로 바이브코딩 맛보기

> 사내 바이브코딩 세미나 자료

## 📋 세미나 개요

AI 코딩 에이전트 **Google Antigravity**를 활용하여 실제 업무에 적용할 수 있는 바이브코딩을 체험합니다.

## 🎯 학습 목표

- Antigravity UI 이해 및 프로젝트 설정
- GEMINI.md를 통한 AI 컨텍스트 관리
- 룰, 워크플로우, 스킬 활용법
- 실습을 통한 바이브코딩 체험

## 📚 세션 구성

### Session 1: 환경 설정 (15분)
1. 프로젝트 디렉토리 생성
2. Antigravity UI 둘러보기
3. GEMINI.md 설정

### Session 2: AI 특징 이해 (15분)
1. 컨텍스트(Context)란?
2. 메모리와 세션 관리
3. 새 세션을 시작하는 이유

### Session 3: 고급 설정 (15분)
1. 룰(Rules) 설정
2. 워크플로우(Workflows) 활용
3. 스킬(Skills) 구성

### Session 4: 실습 (60분+)
1. 쇼핑몰 주간 판매 데이터 보고서 + 시각화
2. API를 이용한 우리 동네 날씨 서비스 구축
